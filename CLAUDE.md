# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 言語設定

**重要**: このプロジェクトでは、Claude Codeとのやり取りは**日本語**で行ってください。コード内のコメント、ドキュメント、コミットメッセージ、およびClaude Codeとの会話はすべて日本語を使用します。

## プロジェクト概要

このリポジトリは現在初期化段階です。プロジェクトの構造とアーキテクチャは開発の進行に伴って更新されます。

## カスタムコマンド

このプロジェクトでは以下のカスタムコマンドが利用可能です：

### `/research <GitHub issue URL | 調査トピック>`
GitHub issueやプロジェクト内のトピックについて調査を実施します。
- GitHub issue のURLが指定された場合: issueのタイトル、本文、コメント、ラベル、マイルストーンを分析
- その他の文字列が指定された場合: プロジェクト内の関連するコードやドキュメントを検索・分析

### `/code-review <ファイルパス | PR番号>`
コードレビューを実施します。
- ファイルパスが指定された場合: そのファイルをレビュー
- PR番号が指定された場合: そのPRの変更をレビュー
- 引数がない場合: 現在のgit diff（ステージングされた変更）をレビュー

## 開発環境

- **GitHub MCP**: GitHub APIへのアクセスが可能に設定されています
- **モデル**: デフォルトはHaikuです
- **サンドボックス**: 有効です
